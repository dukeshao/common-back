### 3.删除用户 - ok

发送请求格式

```js
let url = http://localhost:3355/user/delete
$http(
    url:url,
    method:"DELETE",
    data:{
        ids:[""],//用户 id
    }
)
```

接口返回

```json
{
    "data": {
        "entity": {
            "n": 2,
            "ok": 1
        }
    },
    "msg": "success",
    "status": 200
}
```

```js

    	pagenNum:1,//第几页
```
